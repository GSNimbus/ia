INSTRUÇÕES:
SOLICITAÇÃO DE DADOS PARA FINS DE PROJETOS DE PESQUISAS
- Para solicitação de dados meteorológicos a serem utilizados para fins didáticos e projetos de extensão:

1- Download do arquivo (.zip) contendo um evento para o verão e um evento para o inverno.
Clique aqui para fazer o download de EventoVeraoeInverno.zip

Maiores informações sobre esses eventos entrar em contato com o meteorologista Thiago através do
e-mail: thiago@ipmet.unesp.br


- Para solicitação de dados meteorológicos para fins de projetos de pesquisas efetue os seguintes procedimentos:

1- Leia atentamente as instruções com as informações e opções de dados disponíveis para sua solicitação.
Clique aqui para informações sobre os formatos de dados de radar meteorológico
Clique aqui para informações sobre os dados da estação meteorológica do IPMet-Bauru

2- Download do arquivo MS-Word (.doc) contendo o formulário para preenchimento e assinaturas.
Clique aqui para fazer o download de TermodeCompromisso.doc

3- Leitura e aceite dos termos em que serão solicitados os dados.

4- Preenchimento de todos os campos do "Termo de Compromisso", utilizando o MS-Word.

5- Caso o pedido seja feito pelo aluno, o documento deverá vir assinado também pelo orientador do projeto.

6- Envio do arquivo TermodeCompromisso.doc anexado ao e-mail à Supervisão do IPMet/UNESP, para avaliação preliminar da solicitação. Enviar a solicitação para o e-mail : solicita-dados.ipmet@unesp.br

7- Dados dos radares meteorológicos somente serão cedidos em DBZ e não em taxa de precipitação.
